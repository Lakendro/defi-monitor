## ✨ 特性

- 📈 实时TVL监控
- 💰 代币价格追踪
- 📊 数据可视化
- 📋 自动化报告
- 🔔 价格预警

## 🚀 快速开始

### 安装

```bash
git clone https://github.com/yourusername/defi-monitor.git
cd defi-monitor
pip3 install -r requirements.txt
```

### 使用

```bash
# 运行监控
python3 src/monitor.py

# 生成可视化报告
python3 src/visualizer.py
```
